The website launch

Hello!! So this is the day; I've started this project roughly around the 1st of February, which would mean it took me about 17 days.

Which, honestly, is not at all bad - I mean, I did not know any HTML or CSS before this.

Why?

I've wanted to join the indie web for a long time. Centralised social media has let us create and connect faster than ever, but it did so at the cost of handing over our expression to corporations. It's not even controversial to state that social media makes our lives worse - it's just an accepted fact.

But, in the case of the web... We can just return to how things were. I think this is what pulls me so.

How?

If you're the kind of person to do this, you already know it. But this depends on how you want to do this:

  1. Without programming:

    You can use tools like Google Sites or WordPress to make something with absolutely no code.

    Personally, though, I think you should go with Neocities, and use one of the hundreds of templates other people offer for free.

  2. From scratch:

    Firstly, you'll need to learn how to do HTML/CSS. I recommend starting with the HTML Dog course - It's really easy to follow. After you complete this, do their CSS beginner course, and if you feel up to it - the intermediate ones. This is more than enough to start.

    Next, you need a host - for this, I recommend Neocities again; It's very simple. If you have some know-how, Github Pages is a great alternative.

    And that's it! Now that you have the tools, the web is your canvas.

One last thing: If you choose to do this, and need any help, please ask me. I will help you. For free. I'm not an expert either, but I've clearly managed; We can figure it out together!

Thank you

For stopping by! Feel free to explore the rest of the palace, as I call it; I've archived actual years of my art here.

Cheers!
